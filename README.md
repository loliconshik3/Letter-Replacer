# Описание

Преобразователь текста с русскими символами в текст с частично английскими, без видимых изменений.

# Применение

1. Откройте .bat файл.

2. Введите текст, который необходимо преобразовать.

3. Готово, обработанный текст скопирован в буфер обмена.
